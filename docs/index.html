KEIBA EDGE NAR について

KEIBA EDGE NARは機械学習（XGBoost）を活用した地方競馬（NAR）専用の予想システムです。

約80万件のNAR過去データから騎手成績・コース適性・馬場状態・血統（父馬/母父）など49の特徴量を学習。Optunaチューニング済み。AUC 0.8169 / 5-fold CVで複勝的中率60.5%。◎圧倒レースは自動で3連複◎軸に切替。

大井・川崎・船橋・浦和・門別・名古屋・園田・笠松・金沢・高知・佐賀など全国の地方競馬場に対応。毎日自動で予想を生成し、Discord・Xで配信しています。
